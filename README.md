# ⚾ KBO Data Analysis Project

AI학과 재학 중 데이터 분석 역량 향상을 위해 진행하는 KBO 데이터 분석 프로젝트입니다.

어릴 때부터 관심을 가져온 야구 데이터를 활용하여 Python과 Pandas를 학습하고 있으며, 다양한 분석 프로젝트를 통해 데이터 처리, 시각화, 그리고 인사이트 도출 능력을 키우고 있습니다.

## 🛠 Tech Stack

* Git / GitHub
